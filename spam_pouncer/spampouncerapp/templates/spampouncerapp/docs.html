API Endpoints

Verify Token

POST /verify-token/

Verifies if the provided authorization token is valid.

{
    "token": "your_token_here"
}

Get User Score

POST /get-user-score/

Retrieves the trust score for a specific user ID. Requires valid token.

{
    "token": "your_token_here",
    "user_id": 123
}

Set User Score

POST /set-user-score/

Sets or updates the trust score for a specific user ID. Creates a new user if they don't exist. Requires valid token.

{
    "token": "your_token_here",
    "user_id": 123,
    "score": 42
}

Classify Text

POST /classify-text/

Classifies the provided text for spam detection. If user_id is provided, updates their trust score. Requires valid token.

{
    "token": "your_token_here",
    "text": "text_to_classify",
    "user_id": 123  // Optional
}

Database Schema

Account Model

class Account:
    user_id: Integer (Primary Key)
    name: String (max_length=255)
    trust_score: Integer
    num_updates: Integer
